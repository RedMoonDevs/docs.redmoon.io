Village
=======

## Components
Component | Type | Details
--- | --- | ---
`json` | String | The JSON representation of the Village.
`war` | String | The JSON representation of the WarVillage.
`raw` | String | The JSON given by the data fetched from Supercell. _(The option has to be specified to get this)_
`image` | String | The Pictural representation of the Village. _(The option has to be specified to get this)_
